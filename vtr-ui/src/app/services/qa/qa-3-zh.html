如何了解计算机的 CPU 情况

症状

“32 位”​和“64 位”是指计算机处理器（也称为 CPU）处理信息的方式。64 位版本的 Windows 可比 32 位的系统更高效地处理大量随机存取内存（RAM）。

受影响的品牌

上述症状与以下系统相关，但并不限于以下系统：

  • 所有台式机和笔记本电脑

操作系统

  • Windows 7、8、10

解决方案

Windows 7 或 Vista

  1. 单击“开始”按钮（“开始”按钮图片），然后右键单击“计算机”，最后单击“属性”打开“系统”
  2. 在“系统”下，可查看系统类型。

Windows 8 和 Windows 10/u>

方法 1： 在“控制面板”中查看“系统”窗口

  1. 从屏幕右边缘扫入，然后点击“搜索”。如果使用的是鼠标，请将指针指向屏幕右下角，然后单击“搜索”。
  2. 在搜索框中输入“系统”，然后轻触或单击“设置”。
  3. 轻触或单击“系统”
  4. 如果运行的是 Windows 8 64 位版本，“系统”标题下的“系统类型”字段中将显示“64 位操作系统”。如果运行的是 Windows 8 32 位版本，“系统”标题下的“系统类型”字段中将显示“32 位操作系统”。  

方法 2 : 查看“系统信息”窗口

  1. 从屏幕右边缘扫入，然后点击“搜索”。如果使用的是鼠标，请将指针指向屏幕右下角，然后单击“搜索”。
  2. 在搜索框中，输入“系统信息”
  3. 轻触或单击“系统”，然后轻触或单击“系统信息”，最后单击“系统摘要”。
  4. 如果运行的是 Windows 8 64 位版本，“项目”标题下的“系统类型”字段中将显示“基于 x64 的电脑”。  

    如果运行的是 Windows 8 32 位版本，“项目”标题下的“系统类型”字段中将显示“基于 x86 的电脑”。
