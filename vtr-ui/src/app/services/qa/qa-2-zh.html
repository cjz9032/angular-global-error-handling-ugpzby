如何刷新您的电脑
症状
如果您的电脑性能不如从前却不知道原因，则可以刷新您的电脑，这不会删除​任何个人文件或更改您的设置。Windows 可以“刷新”您的电脑，这就像在不删除个人文件的情况下进行全新安装。
刷新电脑会发生以下结果：您的文件和个性化设置将不会更改
  • 您的电脑设置将恢复为缺省设置
  • 来自 Windows 应用商店的应用程序将会保留
  • 从磁盘或网站安装的应用程序将被删除
  • 已删除文件的列表将保存在您的桌面上
受影响的品牌
  • 所有台式机和笔记本电脑/li>
操作系统
  • Windows 8、8.1 和 10
解决方案
  • Windows 8/8.1
    1. 轻触或单击“设置 ”
    2. 轻触或单击“更新和恢复 ”​
    3. 轻触或单击“恢复 ”
      。在“恢复电脑而不影响你的文件”下 ，轻触或单击“开始”。
    4. 单击“下一步”。.
    5. 请单击“刷新”开始刷新过程。
    6. 刷新过程完成后，可在桌面上找到“已删除的应用”列表
    7. 可以找到刷新系统时删除的应用程序
  • Windows 10
    1. 在“开始”菜单提供的搜索工具中搜索“刷新”，
    2. 在搜索结果中单击“恢复电脑而不影响你的文件”，如下所示。
    3. 单击“恢复电脑而不影响你的文件”​标题下方的“开始”按钮开始刷新您的 Windows 10 电脑。
其他信息
大多数情况下，开始刷新电脑后，它将自己完成该过程。但是，如果 Windows 需要缺少的文件，将会要求您插入恢复盘，通常是 DVD 光盘或 U 盘。如果发生这种情况，所需介质取决于您的电脑。
如果曾使用 DVD 光盘将电脑升级到 Windows 8.1 或 Windows RT 8.1，请使用该光盘。如果没有 Windows 8.1 或 Windows RT 8.1 介质，请联系 Microsoft 支持。
