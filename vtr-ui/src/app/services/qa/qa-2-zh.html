如何重置 Windows 10
如果您的 Windows 10 电脑出现问题或将要出售，则应将其重置。通过执行重置，您可以选择保留或删除文件，删除所有应用程序和设置并安装 Windows 的新副本。这也可以解决诸如性能降低等许多问题。
1.打开“设置”应用程序
按下任务栏的“开始”按钮（通常在左下方），然后选择“设置”按钮。
2.从类别列表中选择“更新和安全”
3.从菜单中选择“恢复”
4.开始重置此电脑
在顶部的“重置此电脑”标题下，单击或点击“开始”按钮。
5.选择是否保留文件
重置后，您可以登录到您的 Microsoft 帐户。使用此帐户购买的所有 Windows 应用商店应用程序都将重新安装。
  • “保留我的文件”会删除您的应用程序和设置，但会保留您的所有个人文件。
  • “删除所有内容”会删除您的应用程序、设置和个人文件。如果没有文件的备份副本，则可能需要按“取消”，将其备份，然后重新开始此过程。
6.您可以选择其他附加设置
仅在上一步中选择“删除所有内容”时，此选项才会显示。
7. 阅读概述，并在准备好进行继续操作时按“重置”。
Windows 将准备重置您的设备。只需等待至您进入蓝屏。
8.当“选择一个选项”出现时，选择“继续”。
Windows 将重新启动并开始重置。重置时，您的电脑可能会重启几次。
社区常见问题解答
如果我选择删除所有内容，它是删除计算机中/连接到我的计算机的所有驱动器，还是仅删除运行 Windows 10 的驱动器？
社区答案
它只会删除 Windows 10 操作系统驱动器（通常为 C盘），其他驱动器将不受影响。
如果屏幕坏了，我可以执行此操作吗？
社区答案
很遗憾，您不能。您需要换一个新屏幕。
如果我已登录任何网站，该历史记录也会被删除吗？
社区答案
是的，具体取决于您使用的浏览器。要删除浏览器历史记录，只需转至浏览器中的设置并删除历史记录即可。您也可以使用隐身模式避免保存历史记录。
如果我选择删除所有内容，是否也会重置密码？
社区答案
是的，一旦您重置 Windows 10 计算机，它将删除所有用户数据。这包括文档、存储的密码、图片、首选设置、浏览器信息和应用程序数据。
